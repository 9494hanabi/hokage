# sample-pj - モーター順回転サンプル

このディレクトリは、FTC環境での開発の最小サンプルです。

## 含まれるファイル

- `SingleMotorForward.java` - ゲームパッドのトリガーでモーター1つを順回転させるTeleOp

## 使い方

1. `SingleMotorForward.java` を `TeamCode/src/main/java/org/firstinspires/ftc/teamcode/` にコピー
2. Android Studio で Build
3. Robot Controller にデプロイ
4. Driver Station で実行

詳しくは `../INSTRUCTION.md` を参照してください。
